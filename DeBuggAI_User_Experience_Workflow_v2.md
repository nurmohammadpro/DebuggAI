## Key UX Decisions (Revised)

| Decision | Rationale |
|----------|-----------|
| **30 free credits on signup** | Enough for 10 debugs + 2 builders — demonstrates value |
| **Team pool credits** | Shared resource encourages collaboration, prevents hoarding |
| **Credit allocation per member** | Prevents one user burning entire team budget |
| **Zero Knowledge toggle** | Privacy as premium feature — differentiates for compliance |
| **Premium actions (50-150 credits)** | Forces heavy users to upgrade or top-up |
| **Top-up never expires** | Reduces purchase friction, locks in revenue |
| **Annual discount (~16%)** | Improves cash flow, reduces churn |
| **Security Launch Package** | Captures compliance-driven buyers at $2,999/year |
| **GitHub/Vercel integrations** | Stickiness — harder to leave once integrated |
| **Admin controls (Enterprise)** | Required for procurement, justifies $999+/mo |
| **14-day team trial** | Low friction for org adoption, proves value before pay |
| **Organization referral ($100)** | Incentivizes B2B viral growth |
